Start the program.

Create a user-defined exception class by extending the Exception class.

In the main class, accept a number from the user.

Check a condition (for example, if the number is negative).

If the condition is true, throw the user-defined exception.

Catch the exception and display an appropriate message.

End the program.

💻 Program:
import java.util.Scanner;

// Step 1: Create user-defined exception
class NegativeNumberException extends Exception {
    public NegativeNumberException(String message) {
        super(message);
    }
}

public class UserDefinedExceptionDemo {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        try {
            System.out.print("Enter a positive number: ");
            int num = sc.nextInt();

            // Step 2: Check condition
            if (num < 0) {
                // Step 3: Throw exception
                throw new NegativeNumberException("Error: Negative numbers are not allowed!");
            }

            System.out.println("You entered: " + num);
        }
        catch (NegativeNumberException e) {
            // Step 4: Handle exception
            System.out.println(e.getMessage());
        }

        sc.close();
    }
}

You entered: 5
